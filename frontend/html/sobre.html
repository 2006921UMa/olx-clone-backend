Sobre o Projeto

Este projeto foi desenvolvido como parte da unidade curricular de Desenvolvimento Web no curso de Técnico Superior Profissional.

O objetivo principal foi criar uma versão funcional e simplificada da plataforma OLX, com um backend completo (autenticação, anúncios, utilizadores, mensagens, avaliações) e um frontend de apoio para testes.

Todo o sistema foi construído com Node.js, Express, Sequelize e MySQL no lado do servidor, e com HTML, CSS e Bootstrap no lado do cliente.
